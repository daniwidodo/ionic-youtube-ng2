# Ionic-youtube-ng2
a youtube client writed in ionic 3 (updated to work with version 3)
* this project is a illustration of the youtube api working with ionic 3.
# Ionic support
    ionic 3

# Installation 
1. sudo npm install -g ionic cordova
2. npm install
3. gulp build
4. ionic platform add android.
